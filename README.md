#  QuickQuiz

Un application web de quiz développée avec Flask dans le cadre du module de programmation web.

##  Description

QuickQuiz est une plateforme de quiz interactive qui permet aux utilisateurs de tester leurs connaissances en développement web à travers 5 catégories : HTML, CSS, Flask, Jinja2 et SQLAlchemy.

## Fonctionnalités

- Inscription et connexion utilisateur
- Dashboard personnalisé avec score total
- 5 catégories de quiz (10 questions chacune)
- Score sur 20 points
- Système de tentatives (3 maximum par catégorie)
- Timer de 5 minutes par quiz
- Résultat avec statut Pass/Fail

##  Technologies utilisées

- Python / Flask
- Jinja2
- SQLite / SQLAlchemy
- HTML / CSS

## Installation

1. Cloner le repository
2. Installer les dépendances :
pip install flask flask-sqlalchemy
3. Créer la base de données :
python questions.py
4. Lancer l'application :
python app.py
5. Ouvrir dans le navigateur :
http://127.0.0.1:5000
